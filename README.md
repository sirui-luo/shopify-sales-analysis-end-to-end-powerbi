# Shopify Sales Analysis | End-to-End Power BI Project

This project is a real-time sales analysis dashboard built using Power BI, leveraging Shopify transactional data. The goal is to deliver actionable insights into sales performance, customer behavior, and retention metrics to support business decision-making.

## Project Objectives

- Analyze sales trends, customer behavior, and retention using Shopify data
- Build an interactive Power BI dashboard with dynamic filters and visuals
- Enable drill-through reporting for detailed transaction-level exploration

---

## Key KPIs Tracked

### Transaction Performance
- Net Sales: Total revenue (before tax)
- Total Quantity: Total items sold
- Net Avg Order Value: Average revenue per order

### Customer Purchase Behavior
- Total Customers
- Single Order Customers
- Repeat Customers

### Retention & Customer Value
- Lifetime Value (LTV)
- Repeat Rate
- Purchase Frequency

---

## Visuals & Features

- Dynamic KPI selector for switching between key metrics
- Province & City Maps: Regional breakdown by sales or customer count
- Daily & Hourly Trend Charts: Understand peak sales periods
- Product Type Analysis: Identify top-selling categories
- Payment Gateway Insights: Analyze usage of payment methods
- Drill-through page: Explore order-level and customer-level detail

---

## Tools & Tech Stack

- Power BI: Data modeling, DAX, and dashboard development
- DAX: Custom calculations for KPIs and dynamic measures
- Excel/CSV: Raw data transformation and validation
- Google Drive: Data source hosting

## Project Steps

1. Requirement Gathering – Defined business goals and KPIs
2. Data Connection – Loaded raw data from external sources
3. Data Cleaning & Quality Check
4. Data Modeling – Created relationships and measures
5. DAX Calculations – Built dynamic metrics and slicers
6. Dashboard Layout – Designed charts and filters for usability
7. Insights Generation – Identified trends and business takeaways

Delivered a scalable, interactive dashboard that empowers decision-makers with clear visibility into revenue drivers, customer loyalty, and purchase patterns.
